Apresentando o front-end do projeto Pet Care

A ideia desse projeto é criar um sistema para gerenciamento de atendimentos veterinários, onde o usuário poderá agendar atendimentos em estabelecimentos veterinários e os profissionais poderão gerenciar seus atendimentos.
Os clientes poderão cadastrar seus pets, realizar agendamentos, visualizar o histórico de atendimentos e avaliar os estabelecimentos.
Após finalizar a versão 1.0 do back-end do projeto Pet Care, chegou a hora de apresentar o front-end.
Contei com a ajuda do colega @wallace para desenhar algumas telas.

Nessa primeira versão foi implementado:

- Tela de login
- Telas de primeiro acesso
- Telas de listagem, cadastro e ediçao de PROFISSIONAIS no perfil de administrador
- Telas de listagem, cadastro e ediçao de ESTABELECIMENTOS no perfil de administrador
- Dashboard do perfil de administrador
- Agenda de atendimentos do perfil de PROFISSIONAL
- Ficha de atendimento do perfil de PROFISSIONAL*


*Ficha de atendimento ainda em desenvolvimento


## Tecnologias utilizadas

- ReactJS
- Typescript
- Styled Components
- React Router Dom
- React Icons
- Axios
- Yup
- tailwindcss
- Material Tailwind
- apexcharts
